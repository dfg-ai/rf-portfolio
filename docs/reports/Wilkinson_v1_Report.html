Wilkinson 功分器 @ 2.44GHz — 项目报告

2026-07-13 | v1 | FR4 (εr=4.4, h=1.6mm) | 正交布局 | 50×60mm

一、这是什么？功率分配器简介

功率分配器 (Power Divider/Splitter) 是射频微波电路中最基础的器件之一。

它的功能很简单：一个输入信号，分成两路（或多路）等功率输出。

应用场景:

• 天线阵列馈电网络 — 把一路信号分给多个天线单元

• 功放并联（平衡放大器）— 两个功放各放大一半，再合路

• 混频器、调制器 — 本振信号分配

• 测量系统 — 参考信号与测试信号分路

Wilkinson 功分器的独特之处:

• 三端口全匹配（S11=S22=S33≈0）

• 输出端口互相隔离（S23≈0）— 一个端口失配不影响另一个

• 低损耗（除了电阻吸收的奇模功率外无额外损耗）

对比: 最简单的 T 型结（三条线直接连在一起）也能分功率，

但两个输出端口会互相干扰——一个端口的反射会传到另一个端口。

Wilkinson 用 λ/4 阻抗变换 + 隔离电阻解决了这个问题。

二、设计目标

中心频率: 2.44 GHz (与贴片天线同频段)

功率分配: 1:1 等分 (Port 1 → Port 2 和 Port 3 各一半功率)

S11 目标: < −10 dB，实际 −16.58 dB

S21 / S31 目标: −3 dB ± 1 dB (理想 −3.01 dB)，实际 −3.67 dB

基板: FR4 (εr=4.4, tanδ=0.02, h=1.6mm，50×60mm)

隔离电阻: 100Ω (v1 跳过，v2 优化加入)

三、设计步骤与原理

Step 1 — 理解核心：为什么需要 λ/4 变换？

如果直接把三个 50Ω 端口接在一起:

Port 1 向分叉点看去 → 两个 50Ω 输出并联 = 25Ω

但 Port 1 是 50Ω → 50Ω ≠ 25Ω → 失配！反射很大！

解决方案: 在分叉点和每个输出端口之间插一段 λ/4 阻抗变换器。

λ/4 线的特性: 一段特性阻抗 Zq、长度 λ/4 的线，

输入阻抗 Zin = Zq² / Zload

Step 2 — 计算变换器阻抗 Zq

从分叉点看: Port 2 支路 = Zq 接 50Ω → Zin = Zq²/50

两个支路并联 = (Zq²/50) / 2 = Zq²/100

需要这个等于 50Ω (输入匹配): Zq²/100 = 50 → Zq = √5000 ≈ 70.7Ω

所以两支路各需一段 70.7Ω、λ/4 长的微带线。

Step 3 — 计算微带线物理尺寸

50Ω 线: 使用微带线公式 → 线宽 W50 = 3.1 mm (FR4, h=1.6mm)

70.7Ω 线: 更高阻抗 → 更窄 → W70 = 1.3 mm

λ/4 长度: λ₀/√εeff/4 = 122.95mm/√3.25/4 ≈ 17.0 mm

其中 εeff ≈ 3.25 是 70.7Ω 微带线的有效介电常数

Step 4 — 隔离电阻的作用

Wilkinson 的巧妙之处: 在两个输出端口之间接一个 100Ω (= 2×Z₀) 的电阻。

偶模 (两输出同相): 电阻两端等电位 → 没有电流流过 → 电阻"不存在"

奇模 (两输出反相): 电阻流过电流 → 吸收奇模功率 → 输出隔离

这就是为什么 Wilkinson 既有好的功率分配，又有好的输出隔离。

v1 暂时跳过电阻——优先验证基本功率分配功能。

Step 5 — 布局设计决策：为什么全正交？

理想 Wilkinson 的两段 λ/4 线是分开一定角度的 (如 ±45°)。

但 HFSS 仿真中发现: 旋转 2D 矩形后，Lumped Port 不再垂直于走线，

导致端口面与走线斜交 → 信号全反射 (S11 ≈ 0 dB)。

解决方案: 全正交布局——所有走线只沿 X 或 Y 方向。

牺牲了优美的 45° 分支，换取了可靠的仿真结果。

★ 这花了 6 轮迭代才找到根因——是最有价值的工程教训。

四、优化历程

v1 | 旋转 45° + 直接重叠 | S11=−0.27 dB

最初设计: 两段 70.7Ω 线从分叉点 ±45° 延伸。

结果: 几乎全反射——功率根本进不去。

v2–v5 | 旋转 + 焊盘 + Unite | S11=−0.27 dB

尝试: 加大重叠区、焊盘、Unite... 全都无效。

→ 问题不在连接，在更深层。

★ 诊断过程 (5 个测试):

test_thru (2端口直线): S11=−35 dB ✓ → 端口格式正确

test_3port (3端口直线): S11=−10 dB → 3端口也 OK

test_diag (2端口斜线): S11≈0 dB → 旋转是问题!

test_tee (T型无旋转): → 锁定: 旋转 = 根因

v6 | 全正交 + Unite | S11=−16.6 dB ✅

去掉所有旋转，横平竖直。输入竖线 → 分叉点 → 两段横线 → 输出竖线。

Unite 所有走线成一个导体 → 完美连接。

★ 根本教训: Lumped Port 的面必须垂直于走线方向。

旋转破坏了垂直关系，端口积分线还在原位，导致全反射。

五、最终结果与评价

[✓] S11: −16.58 dB @ 2.44 GHz — 匹配优秀 (>90% 功率进入)

[✓] S21: −3.67 dB — Port 2 获得约 43% 输入功率

[✓] S31: −3.67 dB — Port 3 获得约 43% 输入功率

[△] S23: −7.59 dB — 输出隔离一般 (无电阻，预期之内)

功率平衡: S21 和 S31 完全一致 (−3.67 dB) → 等分成功 ✓

额外 0.67 dB 损耗 (vs 理想 −3.01 dB):

• FR4 介质损耗 (tanδ=0.02)

• 铜导体损耗 (有限电导率)

• 结不连续性 (线宽从 3.1→1.3→3.1mm 的变化)

六、面试要点（三句话）

1. 设计了一个 2.44GHz 等分 Wilkinson 功分器：核心是 Zq=70.7Ω λ/4 阻抗变换，

把分叉点的 25Ω 匹配到输入 50Ω。

2. 结果: S11=−16.6dB，S21=S31=−3.67dB (功率完美等分)，

与上一个贴片天线同频(2.44GHz)同基板(FR4)，形成系列作品。

3. 关键收获: Wilkinson 的 3dB 功分来自 λ/4 变换，隔离来自 100Ω 电阻；

HFSS 仿真时 Lumped Port 必须垂直于走线——这是花了 6 轮迭代学到的工程直觉。
