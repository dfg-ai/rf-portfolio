端耦合带通滤波器 @ 2.44GHz — 项目报告

2026-07-14 | v1 | FR4 (h=1.6mm) | 2-pole Chebyshev | 50×90mm

一、这是什么？滤波器简介

滤波器是射频系统三大基础器件之一（天线、功分器、滤波器）。

它的作用：让想要的频率通过，阻挡不想要的频率。

类比：滤波器像一扇"频率门"——只允许特定频率范围的信号通过。

为什么重要？

• 避免干扰：WiFi 用 2.4-2.48GHz，滤波器阻挡带外信号

• 射频前端必备：每个收发机都有滤波器

• 面试中"天线+功分+滤波"三件套 = 完整链路理解

二、设计目标

中心频率: 2.44 GHz

类型: 2阶 Chebyshev 带通 (0.5dB纹波), 5%分数带宽

拓扑: 端耦合半波长谐振器 (全竖线，正交布局)

S11 目标: < −10 dB | 实际: −13.16 dB @ 2.42 GHz

S21: −5.85 dB (FR4 介质损耗限制)

三、设计步骤与原理

1. 为什么选端耦合拓扑？

平行耦合线需要旋转 45° → 踩昨天的坑。

端耦合 = 全竖线排列，间隙耦合，天然正交。

2. 结构: Port1─[Feed]─g01─[R1 λ/2]─g12─[R2 λ/2]─g23─[Feed]─Port2

信号通过间隙逐级耦合 → 仅谐振频率附近能通过 → 带通滤波

3. Chebyshev 原型 (0.5dB ripple):

g₀=1.0, g₁=1.5963, g₂=1.0967, g₃=1.5963

外部 Q = 31.93, 耦合系数 k = 0.0378

4. 物理尺寸:

50Ω线宽=3.1mm, 谐振器=30.8mm (λ/2)

g01=g23=0.15mm (外部耦合), g12=0.6mm (谐振器间)

四、优化历程

v1 L=33.3 g01=0.3 g12=0.5 | f₀=2.31 GHz (偏低)

v2 L=31.5 g01=0.15 g12=0.3 | f₀=2.39 GHz

v3 L=29.8 g01=0.15 g12=0.6 | f₀=2.52 GHz (过冲)

v4 L=30.8 g01=0.15 g12=0.6 | f₀≈2.42 GHz ✅

★ 端耦合滤波器对间隙极度敏感 (0.1mm级)

FR4 的 tanδ=0.02 是 S21 偏高的主因

五、最终结果

[✓] S11: −13.16 dB — 带通响应明确

[✓] 频率: ~2.42 GHz, 接近目标

[△] S21: −5.85 dB — FR4 材料限制

三个项目共性: FR4 廉价但损耗高 → 换 Rogers 可大幅改善

六、面试要点（三句话）

1. 设计了一个 2.44GHz 端耦合带通滤波器：Chebyshev原型→耦合系数→物理尺寸，

全正交布局避免旋转陷阱，5轮迭代优化间隙和长度。

2. 2阶响应，S11=−13dB，可与贴片天线+功分器组成完整射频前端链路。

3. 关键理解：间隙控制带宽、长度控制频率；FR4插入损耗偏高是材料选择而非设计缺陷。
