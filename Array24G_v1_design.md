# 24GHz 2×2 微带阵列 — 设计计算

> Rogers RT/duroid 5880: εr=2.2, tanδ=0.0009, h=0.508mm

## 1. 单个贴片单元

对比 2.4GHz vs 24GHz:

| 参数 | 2.44GHz (FR4) | 24GHz (Rogers 5880) |
|------|-------------|-------------------|
| 贴片 W | 37.4 mm | **4.94 mm** |
| 贴片 L | 28.6 mm | **3.89 mm** |
| 50Ω 线宽 | 3.1 mm | **1.55 mm** |
| λ/4 @ f₀ | 17 mm | **~2.3 mm** |
| 基板 h | 1.6 mm | **0.508 mm** |

全部等比缩小约 7.5 倍。

## 2. 阵列参数

| 参数 | 值 | 理由 |
|------|-----|------|
| 阵元间距 dx=dy | **7 mm** | ≈ λ₀/2 (6.25mm)，略大避免互耦过强 |
| 阵元数 | 2×2 = 4 | 入门示范 |
| 馈电方式 | 并联等分 | 3个T型功分器（70.7Ω λ/4匹配） |

## 3. T 型功分器设计

同 Wilkinson 原理（不加隔离电阻）:
- 输入 50Ω → 分叉 → 25Ω
- 两路各 λ/4 70.7Ω → Zin=100Ω
- 并联 = 50Ω ✓

70.7Ω线宽: 0.6mm, λ/4长度: ≈2.3mm

## 4. 布局 (全正交)

```
       E3 ◻              ◻ E4
         │                │
    ┌────┴──┐        ┌───┴────┐
    │ T_L   │        │  T_R   │
    └────┬──┘        └───┬────┘
         │               │
         └───────┬───────┘        ← 上半层
                 │
             ┌───┴───┐            ← T_C (中心T结)
             │  T_C  │
             └───┬───┘
                 │                ← 下半层
         ┌───────┴───────┐
         │               │
    ┌────┴──┐        ┌───┴────┐
    │ T_LB  │        │  T_RB  │
    └────┬──┘        └───┬────┘
         │                │
       E1 ◻              ◻ E2
         │                │
       Port1
```

实际上 2×2 阵列的馈电网络是 3 个 T 型结的级联，输入到 4 个输出等路径长度。

## 5. 基板尺寸

30mm × 30mm (够放下 4 个贴片 + 馈电网络)
