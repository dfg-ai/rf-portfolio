# 矩形微带贴片天线 @ 2.44GHz — 项目报告

> 项目日期: 2026-07-09 | 版本: v1 (最终 Lp=28.6mm)

## 1. 设计目标

| 参数 | 目标 |
|------|------|
| 项目类型 | 矩形微带贴片天线 (Rectangular Microstrip Patch Antenna) |
| 中心频率 | 2.44 GHz (WiFi) |
| 带宽 | ≥ 80 MHz (2.40-2.48 GHz) |
| S11 | < -10 dB @ 2.44 GHz |
| 基板 | FR4 (εr=4.4, tanδ=0.02, h=1.6mm) |

## 2. 设计方法

### 2.1 基板
| 参数 | 值 |
|------|-----|
| 材料 | FR4_epoxy |
| εr | 4.4 |
| tanδ | 0.02 |
| 厚度 | 1.6 mm |
| 尺寸 | 47 × 39 mm |

### 2.2 初始尺寸（传输线模型）

| 参数 | 公式 | 计算值 |
|------|------|--------|
| 贴片宽度 W | \(W = \frac{c}{2f_0}\sqrt{\frac{2}{\varepsilon_r+1}}\) | 37.4 mm |
| 有效介电常数 εeff | 传输线模型 | 4.08 |
| 延伸长度 ΔL | 边缘场修正 | 0.74 mm |
| 贴片长度 L | \(L = \frac{c}{2f_0\sqrt{\varepsilon_{eff}}} - 2\Delta L\) | 29.0 mm |

### 2.3 内嵌馈电设计

贴片边缘阻抗 ~200Ω，通过内嵌深度 y₀ 匹配到 50Ω：

$$50 = 200 \cdot \cos^2\left(\frac{\pi y_0}{28}\right) \Rightarrow y_0 = 9.3\text{ mm}$$

- 凹槽宽: 6.1 mm (馈线 3.1mm + 两侧间隙 2×1.5mm)
- 凹槽深: 9.3 mm

## 3. 优化历程

| 版本 | Lp (mm) | f₀ (GHz) | S11 (dB) | 方法 |
|------|---------|----------|----------|------|
| v1 | 29.0 | 2.36 | -6.4 | 初始设计，直接馈电 |
| v2 | 28.0 | 2.44 | -6.8 | 缩短贴片，仍未匹配 |
| **v3** | **28.6** | **2.44** | **-19.1** | 内嵌馈电 ✅ |

> 关键教训：调贴片长度只修正频率；**阻抗匹配需要内嵌馈电**。

## 4. 最终结果

| 指标 | 值 | 评价 |
|------|-----|------|
| 谐振频率 | 2.44 GHz | 🟢 精准命中 |
| S11 @ f₀ | -19.15 dB | 🟢 匹配优秀（反射 <1.2%） |
| -10dB 带宽 | 2.41-2.46 GHz (50 MHz) | 🟡 刚好覆盖 WiFi 2.4G |
| 峰值增益 | 2.0 dBi | 🔴 偏低（FR4 损耗高） |

### 增益偏低的原因

FR4 的 tanδ=0.02 在 2.4GHz 导致介质损耗大。正常微带天线增益 5-7 dBi。若换 Rogers 4350B (tanδ=0.0037)，增益可提升到 5+ dBi。

## 6. 面试要点

> 3 句话介绍：

1. **做了什么**: 设计了一个 2.44GHz 矩形微带贴片天线，使用传输线模型计算初始尺寸，内嵌馈电实现阻抗匹配
2. **关键数据**: S11 = -19.15 dB，-10dB 带宽 50 MHz，50mm×39mm 的小尺寸
3. **技术亮点**: 理解了贴片长度控制谐振频率、内嵌馈电控制阻抗匹配的分离设计方法；用 FR4 作为低成本的妥协方案，清楚知道换 Rogers 材料后增益可从 2dBi 提升到 5+dBi
