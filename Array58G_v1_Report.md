# 5.8GHz 4×4 串-并混合阵列 — 项目报告

> 2026-07-18 | v1 | FR4 (h=1.6mm) | 串馈列 + 并馈行 | 5.8GHz

## 1. 设计目标

| 参数 | 目标 | 实际 |
|------|------|------|
| 频率 | 5.8 GHz | 5.73 GHz (−1.2%) |
| S11 | < −10 dB | −18.2 dB |
| 增益 | > 8 dBi | 8.4 dBi |
| 波束 | 铅笔形双向窄 | ✅ X/Y 双向窄 |
| 拓扑 | 4×4 串并混合 | ✅ |

## 2. 阵列拓扑

4 列串馈 × 4 行：每列是独立的 1×4 串馈线阵，4 列通过并联馈电网路统一激励。

```
串馈（列内）：Port1 → 贴片1 → 贴片2 → 贴片3 → 贴片4（λg间距=25mm）
并馈（列间）：T型功分网路，4路等分 → 4列
```

## 3. 关键参数

| 参数 | 初始 | 优化后 |
|------|------|--------|
| 贴片长度 Lp | 11.75mm | **10.50mm** |
| 阵间距 λg | 28.0mm | **25.0mm** |
| 内嵌深度 Gd | 3.92mm | **3.50mm** |
| 列间距 dx | 26mm | 26mm |

## 4. 优化历程

v1→v2: 全局缩放 5.5% → 中心 5.57GHz
v3: 二次缩放 → 5.70GHz  
v4: 三次缩放 → **5.73GHz** ✅

## 5. 对比：单贴片 → 1×4 → 4×4

| 规模 | 增益 |
|------|------|
| 单贴片 5.8GHz | ~5 dBi |
| 1×4 串馈线阵 | 4.1 dBi |
| **4×4 阵列** | **8.4 dBi** |

## 6. 面试要点

1. 设计了一个 5.8GHz 4×4 串-并混合阵列：串馈列（λg间距同相）+ 并馈行（等路径功分）
2. 增益 8.4dBi，波束铅笔形窄波束，S11=−18dB
3. 串馈核心：间距=λg保证同相；混合拓扑 = 真实相控阵的简化版
