# Wilkinson 功分器 @ 2.44GHz — 项目报告

> 2026-07-13 | v1 | FR4, h=1.6mm | 正交布局

## 1. 设计目标

| 参数 | 目标 |
|------|------|
| 中心频率 | 2.44 GHz |
| S11 | < −10 dB |
| S21 / S31 | −3 dB ± 1 dB（等分） |
| 基板 | FR4 (εr=4.4, tanδ=0.02, h=1.6mm) |

## 2. 设计方法

### 2.1 原理

Wilkinson 功分器由两段 λ/4 阻抗变换器组成：

- 分叉点处两个 50Ω 支路并联 = 25Ω
- 70.7Ω λ/4 变换器：Zin = Zq² / Zload = 70.7² / 50 = 100Ω
- 两个 100Ω 并联 = 50Ω → 输入匹配

### 2.2 设计参数

| 参数 | 值 | 说明 |
|------|-----|------|
| 50Ω 线宽 | 3.1 mm | FR4 微带线 |
| 70.7Ω 线宽 | 1.3 mm | Zq = 50×√2 |
| λ/4 长度 | 17 mm | @ 2.44GHz, εeff≈3.25 |
| 隔离电阻 | 跳过（v2） | 100Ω, 需精确布局 |

### 2.3 布局设计

采用**全正交布局**（横平竖直，无旋转），避免端口面与走线不垂直导致的耦合失败：

```
            Port3         Port2
              │              │
              │ 50Ω (竖)     │ 50Ω (竖)
              │              │
    ┌─────────┴──────────────┴─────────┐
    │ 70.7Ω λ/4 (横)  70.7Ω λ/4 (横)  │
    └─────────┬──────────────┬─────────┘
              │  50Ω (竖)    │
              │             Port1
```

**关键教训**：IronPython API 中旋转（Rotate）会破坏 Lumped Port 与走线的垂直耦合——这是反复调试后发现的核心问题。

## 3. 优化历程

| 版本 | 布局 | S11 | S21/S31 | 关键发现 |
|------|------|-----|---------|----------|
| v1 | 旋转 45°+ 直接馈电 | −0.27 dB | −63 dB | 旋转导致端口失配 |
| v2-v5 | 旋转 + 焊盘 + Unite | −0.27 dB | −63 dB | 旋转是根本原因 |
| **v6** | **正交布局 + Unite** | **−16.6 dB** | **−3.67 dB** | ✅ 旋转才是根因 |

> 关键教训：**HFSS Lumped Port 要求端口面与走线垂直**。旋转走线导致端口面与走线斜交，耦合全断。

## 4. 最终结果

| 指标 | 目标 | 实际 | 评价 |
|------|------|------|------|
| 谐振频率 | 2.44 GHz | 2.44 GHz | 🟢 |
| S11 | < −10 dB | **−16.58 dB** | 🟢 匹配优秀 |
| S21 | ≈ −3 dB | **−3.67 dB** | 🟢 等分，±1dB 内 |
| S31 | ≈ −3 dB | **−3.67 dB** | 🟢 左右一致 |
| S23 | — | **−7.59 dB** | 🟡 无隔离电阻 |

### 插入损耗分析

额外 0.67 dB 损耗来源：
- FR4 介质损耗（tanδ=0.02）
- 铜导体损耗
- 结不连续性

隔离电阻（100Ω）将在 v2 版本加入，解决与焊盘重叠导致性能退化的问题。

## 5. 结论

- [x] S11 优秀（−16.58 dB）
- [x] 功率等分（S21=S31=−3.67 dB）
- [x] 与贴片天线同频段（2.44 GHz），形成系列
- [ ] 隔离度待改善（v2 加电阻）

## 6. 面试要点

> 三句话：

1. 设计了一个 2.44GHz Wilkinson 功分器：λ/4 70.7Ω 阻抗变换，全正交布局避免端口耦合问题
2. S11=−16.6dB，功率等分 −3.67dB/每路，与贴片天线项目同频段同基板
3. 关键理解：Wilkinson 的核心是 λ/4 阻抗变换 + 隔离电阻，调试中发现 Lumped Port 与走线的垂直关系是关键约束
