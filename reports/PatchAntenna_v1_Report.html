矩形微带贴片天线 @ 2.44GHz — 项目报告

2026-07-09 | v1 (Lp=28.6mm) | FR4 (εr=4.4, h=1.6mm) | 47×39mm

一、这是什么？微带贴片天线简介

微带贴片天线 (Microstrip Patch Antenna) 是最常见的平面天线之一。

结构：一层薄金属"贴片"放在介质基板上，基板下面是接地平面。

工作原理（直觉版）：贴片相当于一个半波长谐振器——电磁波在贴片两端的

开放边缘处"泄漏"出去，形成辐射。可以理解为：两块水平放置的金属板之间

的电场，在边缘处"漏"了出来。

为什么重要？

• WiFi 路由器、手机、GPS 设备里大量使用

• 低剖面（薄）、轻、易制造（PCB 工艺）

• 天线岗面试第一高频考题

• 阵列天线的基本单元——会做贴片 = 能做阵列

二、设计目标

中心频率: 2.44 GHz (WiFi 第 1 频道)

S11 目标: < −10 dB (反射功率 < 10%)，实际 −19.15 dB (反射 < 1.2%)

带宽目标: 覆盖 2.40–2.48 GHz (80 MHz)，实际 50 MHz (刚好够)

基板: 低成本 FR4 (εr=4.4, tanδ=0.02, h=1.6mm)

馈电方式: 微带线 + 内嵌馈电

三、设计步骤与原理

Step 1 — 计算贴片宽度 W

公式: W = c / (2f₀) × √[2/(εr+1)]

W = (3×10¹¹) / (2×2.44×10⁹) × √(2/5.4) = 61.48 × 0.6086 = 37.4 mm

W 决定了天线的输入阻抗和带宽。W 越宽 → 带宽越宽，但太大

会激励高次模（不想要的谐振）。37.4mm 在合理范围内。

Step 2 — 计算有效介电常数 εeff

电磁波一部分在介质中、一部分在空气中传播，所以"有效"介电常数

介于 εr 和 1 之间:

εeff = (εr+1)/2 + (εr−1)/2 × [1+12h/W]^(-1/2) = 4.08

Step 3 — 计算边缘场修正 ΔL

贴片边缘的电场会"溢出"到空气中（边缘场效应），等效于贴片变长了一点。

ΔL = 0.412h × [(εeff+0.3)(W/h+0.264)] / [(εeff−0.258)(W/h+0.8)] = 0.74 mm

→ 物理贴片需要比"电长度"短 2×ΔL，才能谐振在正确频率。

Step 4 — 计算贴片长度 L (最关键的尺寸)

L = c / (2f₀√εeff) − 2ΔL = 30.43 − 1.478 = 28.95 mm

贴片长度 ≈ λ/2 (在介质中) ≈ 30.4mm。减去边缘场修正后 ≈ 29mm。

最终优化到 Lp = 28.6mm 时谐振在 2.44 GHz。

Step 5 — 内嵌馈电 (Inset Feed)

问题: 贴片边缘的输入阻抗约 200–300Ω，而馈线是 50Ω。

直接接上去 → 严重失配 (S11 只有 −6.4 dB)。

解决: 把馈线"伸进"贴片内部。贴片内部的电压呈余弦分布，

越往里阻抗越低:

Rin(y₀) = R_edge × cos²(π × y₀ / L)

50 = 200 × cos²(π × y₀ / 28) → y₀ = 9.3 mm

在贴片底边切一个 6.1mm×9.3mm 的凹槽，馈线伸入 9.3mm。

这个凹槽的位置控制阻抗匹配。

四、优化历程

v1 | Lp=29.0mm | f₀=2.36 GHz | S11=−6.4 dB

初始尺寸：传输线模型是近似公式，f₀ 偏低 3% 很正常。

S11 差是因为馈线直接接边缘，200Ω 对 50Ω 严重失配。

v2 | Lp=28.0mm | f₀=2.44 GHz | S11=−6.8 dB

缩短贴片 → 频率正确。但阻抗还是不匹配！

→ 发现: 调长度只能修正频率，不能解决匹配问题。

v3 | Lp=28.6mm | f₀=2.44 GHz | S11=−19.1 dB ✅

加入内嵌馈电 → 频率和匹配同时优秀。

★ 核心认知: 贴片长度控制"频率"，内嵌深度控制"匹配"

——两者几乎独立，这是微带天线的核心设计规律。

五、最终结果与评价

[✓] 谐振频率: 2.44 GHz — 精准命中

[✓] S11 @ f₀: −19.15 dB — 反射功率 < 1.2%，匹配优秀

[✓] −10dB 带宽: 50 MHz (2.41–2.46 GHz) — 刚好覆盖 WiFi 2.4G

[△] 峰值增益: 2.0 dBi — 偏低

增益为什么低？

FR4 的损耗角 tanδ=0.02 在 2.4GHz 远高于 Rogers 4350B (tanδ=0.0037)。

大量电磁能量在介质中变成热量，而不是辐射出去。

这不是设计错误——是材料选择。

面试时主动讨论这一点 = 展示工程判断力。
